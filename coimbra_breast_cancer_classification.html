Breast Cancer Classification¶

A K-NN Application¶

by Dave Emmanuel Q. Magno and Aries Valeriano

Executive Summary¶

The goal of this project is to build a prediction model which take features such as, Age, BMI, Glucose, Insulin, HOMA, Leptin, Adiponectin, Resistin, MCP.1 as predictors to predict presence or absence of breast cancer. These predictors are anthropometric data and parameters which can be gathered in routine blood analysis.

To achieve this goal, the process are based on the whole cyle of data life, which starts from data processing up to predictive modelling. For reproducibility, syntax are shown with some comments to have clear understanding with the process.

In data processing, descriptive features are normalized to make sure all of them has equal contribution to the model trained.

In the data exploration, there are findings that seems to give some useful information, such as the distribution of every descriptive features, correlation between pairs of descriptive features, as well as between every descriptive feature and the target feature. It is then revealed through trying different combination of descriptive features during model training that only Resistin, Glucose, Age, and BMI produced the highest accuracy. Hence, irrelevant descriptive features were remove to reduce training time, improve model, and reduce overfitting.

In predictive modelling, 10-fold cross validation based on k-nearest neighbors algorithm with hyperparameter k ranges from 1 to 35 repeated 5 times was performed. Grid search then was used to find the optimal value of k and it was found out that at k=7 the accuracy of the model is as its highest having 78.04%. This only imply that the model can predict the presence or absence of breast cancer with 78.04% accuracy.

# Scale features using MinMaxScaler
def scale_feat(X: pd.DataFrame,
               mode=None) -> pd.DataFrame:
    if mode == None:
        mode = MinMaxScaler()
    _ = mode.fit(X)
    _ = _.transform(X)
    return pd.DataFrame(_, columns=X.columns)

# Hyperparameter optimization using GridSearch
def tune_knn(X: pd.DataFrame, y: pd.DataFrame,
             k_range: np.ndarray=None,
             param_grid: dict=None, cv: int=None):
    if k_range == None:
        k_range = np.arange(1, 35)
    if param_grid == None:
        param_grid = {
            'n_neighbors': k_range,
            'weights': ['uniform', 'distance'],
            'metric': ['euclidean', 'manhattan']
        }
    cv = RepeatedKFold(n_splits=10, n_repeats=5, random_state=7)
    clf = GridSearchCV(estimator=KNeighborsClassifier(),
                       param_grid=param_grid,
                       scoring='accuracy',
                       cv=cv,
                       verbose=1,
                       n_jobs=-1)
    clf.fit(X, y)
    return clf

# Testing best model
def test_knn(X_test: pd.DataFrame,
             y_test: pd.DataFrame,
             tuned_model) -> float:
    clf = tuned_model.best_estimator_
    predictions = clf.predict(X_test)
    score = accuracy_score(y_test, predictions)
    return score

# Show results on training and validation
def show_knn_results(best_params: dict, best_score: float,
                     mean_test_score: np.ndarray):
    print("Best model parameters: ")

    print("   weights: ", best_params['weights'])
    print()
    print("Best average score: ", round(best_score*100, 2), "percent")
    plt.plot(mean_test_score)

Data Description¶

The dataset comes from Breast Cancer Coimbra DataSet. It is a multivariate dataset with 116 samples and have the following 10 attributes:

Quantitative:

  • Age:
    • The age of the respondent
  • BMI:
    • Body mass index measured by kg/m2
  • Glucose:
    • Simple sugar from the body of the respondent measured by mg/dL
  • Insulin:
    • A peptide hormone produced by beta cells of the pancreatic islets measured by µU/mL
  • HOMA
    • Homeostasis model assessment
  • Leptin:
    • A hormone predominantly made by adipose cells and enterocytes in the small intestine that helps to regulate energy balance by inhibiting hunger, which in turn diminishes fat storage in adipocytes. Measured by ng/mL
  • Adiponectin:
    • A protein hormone and adipokine measured by µg/mL.
  • Resistin:
    • A cysteine-rich peptide hormone measured by ng/mL
  • MCP-1:
    • Monocyte chemoattractant protein-1 measured by pg/dL

Qualitative:

  • Classification:
    • 1 = Healthy controls (or absence of breast cancer)
    • 2 = Patients (or presence of breast cancer)

The objective is, knowing the descriptive features such as Age, BMI, Glucose, Insulin, HOMA, Leptin, Adiponectin, Resistin, MCP-1, build a model that can predict the presence or absence of breast cancer.

These dataset will be split into training set which contains 92 samples and testing set which contains 24 samples. Training set is used for building a model and testing set will be used for assessing the accuracy of the model built from training set.

From this, we now have idea that for every pair mentioned, if 1 of it say, "Glucose" is correlated to the target feature, then possibly the other one say, "HOMA" is also correlated with the target feature.

Data Pre-processing¶

Feature selection¶

In selecting the necessary features relevant to predict the absence or presence of breast cancer, we follow the results of the paper, "Using Resistin, glucose, age and BMI to predict the presence of breast cancer." The dataset used in this project was also taken from the authors of this paper. Based on their results, the presence of breast cancer in women could be predicted on a test data set with sensitivity ranging between 82 percent and 88 percent, and specificially ranging between 85 percent and 90 percent (95 percent CI for the AUC is [0.87, 0.91]). These suggests that considering only resistin, glucose, age, and BMI as predictive features may classify the target well.

In addition, we tried different combination of features but the combination with the highest accuracy can be achieved by selecting resistin, glucose, age, and BMI. Moreover, we also tried to include all features in the dataset, but unfortunately, the accuracy is low, both on the training and testing set. Therefore, based on these observations, we decided to only consider resistin, glucose, age, and BMI as the features of our model.

Conclusion and Recommendations¶

In this project, we utilized the k-Nearest Neighbors algorithm in classifying the absence or presence of breast cancer in patients. We built a prediction model which takes resistin, glucose, age, and BMI as predictive features to determine whether a patient has a breast cancer or not. The dataset used in this study was the Breast Cancer Coimbra dataset.

Fortunately, the dataset has no missing values. We explored the data and found that the feature variables doesn't seem to have a Normal distribution. In addition, the correlation matrix produced using Spearman shows that the features, insulin and HOMA, have the highest pairwise correlation coefficient.

Moreover, after feature selection 4 descriptive features included in the model. The model still was validated using all the available features in the dataset however, the model with only 4 predictive features produce much higher accuracy.

We tuned the necessary parameters of our model and got the best parameter setting of using manhattan as metric, distance as weights, and k equal to 7. Using these settings, our model achieved an accuracy score of 78.04 percent in the validation stage. This shows that the trained model can predict the absence or presence of breast cancer in patients with an accuracy of about 78.04 percent.

Further researches may use Principal Component Analysis in selecting the features and standardization may be applied in feature scaling.
